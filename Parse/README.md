# Setup

1. Install Parce CLI
2. Clone Repo ```git clone git@github.com:bvallelunga/EnjoyPnD.git```
3. Make project your default ```parse default enjoypnd```

# Develop Code

```
parse develop enjoypnd
```

# One Time Deploy(s)

```
parse deploy
```
